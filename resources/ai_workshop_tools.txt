AI Workshop Tools and Resources
===============================

LLM and AI Services
------------------
- OpenAI API: https://platform.openai.com/ - For GPT models
- Hugging Face: https://huggingface.co/spaces - Free LLMs like Mistral-7B, BLOOM
- Ollama: https://ollama.ai - Run LLaMA 3/Mistral locally
- Google AI Studio: https://aistudio.google.com/ - For Gemma models
- Anthropic Claude: https://console.anthropic.com/ - Claude API

Prompt Engineering
-----------------
- Learn Prompting: https://learnprompting.org - Comprehensive guide
- Awesome ChatGPT Prompts: https://github.com/f/awesome-chatgpt-prompts

Image Generation
---------------
- DALL-E API: https://openai.com/dall-e-3 - OpenAI's image generation
- Stability AI: https://stability.ai/platform - Stable Diffusion
- Craiyon: https://www.craiyon.com - Free image generation

Development Environments
----------------------
- Google Colab: https://colab.research.google.com - Free GPU/TPU access
- Kaggle Kernels: https://www.kaggle.com/kernels - Free notebooks with competitions
- Replit: https://replit.com - Browser-based IDE

Key Libraries and Frameworks
--------------------------
- LangChain: https://python.langchain.com - Orchestrate prompts and retrieval
- Transformers: https://huggingface.co/docs/transformers/index - Hugging Face
- Sentence Transformers: https://www.sbert.net - State-of-the-art embeddings
- ChromaDB: https://docs.trychroma.com - Vector database
- FAISS: https://github.com/facebookresearch/faiss - Similarity search
- PyPDF2: https://pypi.org/project/PyPDF2 - Extract text from PDF files
- PEFT: https://github.com/huggingface/peft - Parameter-Efficient Fine-Tuning
